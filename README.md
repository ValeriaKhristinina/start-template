# Template for start project

Terminal:
1. git clone (link);
2. rename directory;
3. Delete .git-file (rm -rf .git);
4. git init;

First install dependencies:
```npm install```

### Develop:
```gulp dev```
This command run server on [http://localhost:3001/](http://localhost:3001/) and watching changes
"ctrl+C" stop server

### Build
```gulp build```
Build project for production
